### Added

- Initial release
- 5 built-in reviewer personas: security, performance, readability, architecture, testing
- Multi-round adversarial debate between agents
- Consensus scoring with endorsements, challenges, and withdrawals
- Swarm summary showing agreement, splits, and emergent findings
- Custom persona support via YAML files (`.dissent.yaml` or `--persona-file`)
- Rich terminal, JSON, and Markdown output formats
- OpenAI, Ollama, and any OpenAI-compatible API support
- CLI with `dissent` command
